# 客户端安装指南 :id=top

客户端最新版本号: `V1.2.0`

## 第一次下载客户端 :id=first

[点击](/download)进入下载中心

下载`黑鸽院完整客户端.zip`到本地一个**你喜欢的位置**并解压

单击`HMCL.exe`即可打开启动器

登陆你的**正版账号**并启动即可

?> `HMCL.exe`为游戏启动器  
`WebApp/黑鸽院.exe`为网页客户端  
`点击更新.exe` / `update.exe`为更新工具

## 更新客户端 :id=update

!> 如果你还没有下载客户端，请前往查看 [第一次下载客户端](#first)

请先对比你当前客户端的版本号和最新客户端版本号是否一致, 如果不一致, 点击`点击更新.exe`或者`update.exe`即可更新

- 你可以通过查看本地文件中的 **当前版本Vx.x.x** 获取**当前客户端的版本号**

- 你可以在[首页](Home.md) / 这个页面的[顶部](#top)获取**最新客户端的版本号**

![McAppHelp-3.webp](https://heigeyuan-web.oss-accelerate.aliyuncs.com/images/McAppHelp-1.webp)

!> **注意: 请勿频繁更新, 请在确保你当前客户端的版本和最新客户端的版本不一致的情况下更新**
